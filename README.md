# ZeroClaw Home Assistant Add-ons

Custom Home Assistant Add-on repository for ZeroClaw.

## Add repository to Home Assistant

Use this repository URL in Home Assistant Add-on Store:

```text
https://github.com/SeoFood/hassio-addons-zeroclaw
```

## Included Add-ons

- `zeroclaw`: Run the ZeroClaw daemon as a Home Assistant add-on.
